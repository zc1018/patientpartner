# Value Score Calculator - 价值分进度计算器

用于计算各系统价值分 L1/L2/L3 进度并生成月度报告。

## 功能说明

根据系统价值分现状、目标和当月实际数据，计算各层级进度和总进度。

### 计算公式

```
进度 = (当月值 - 现状值) / (目标值 - 现状值) × 100%

总进度 = (L1进度 + L2进度 + L3进度) / 3
```

### 输入文件

1. **价值分通晒表格.xlsx** - 主跟踪表格
   - 初始值 sheet: 包含各系统的现状值和目标值
   - 历史月份 sheets: 每月的价值分数据

2. **系统价值分统计.xlsx** - 当月原始数据
   - 包含各系统的 L1商业价值分、L2渗透率、L3系统分

## 使用场景

### 场景1：生成新月度sheet

**用户说：**
```
根据3月2日的数据创建3月的sheet
```

**AI执行：**
1. 读取价值分通晒表格.xlsx 的初始值 sheet 获取现状和目标
2. 读取系统价值分统计.xlsx 获取当月数据
3. 计算 L1/L2/L3 进度和总进度
4. 新建月份 sheet 并写入数据
5. 格式化进度列为百分比

### 场景2：计算特定系统进度

**用户说：**
```
帮我计算A择校的L1进度，现状120，目标130，当月值125
```

**AI执行：**
```
进度 = (125 - 120) / (130 - 120) = 5/10 = 50%
```

### 场景3：批量更新进度

**用户说：**
```
更新所有系统的2月进度
```

**AI执行：**
1. 读取系统价值分统计表
2. 匹配各系统数据
3. 计算所有系统进度
4. 更新2月sheet

## 输出格式

生成的月度sheet包含以下列：

| 列名 | 说明 |
|-----|------|
| 系统 | 系统名称 |
| 车厢 | 业务域 |
| 产品 | 产品负责人 |
| L1现状/2月/L1目标/L1进度 | L1商业价值分相关 |
| L2现状/2月/L2目标/L2进度 | L2渗透率相关 |
| L3现状/2月/L3目标/L3进度 | L3系统分相关 |
| 总进度 | 三项进度平均值 |

## 进度解读

- **进度 > 100%**: 超额完成目标
- **进度 = 100%**: 正好达成目标
- **进度 = 0%**: 与现状持平，无进展
- **进度 < 0%**: 相比现状退步

## 注意事项

1. 系统名称映射需准确，否则会找不到对应数据
2. 当月数据为NaN时，进度显示为空
3. 现状值=目标值时，进度按1.0（100%）或0.0计算
4. 总进度为三项进度的平均值（排除空值）
