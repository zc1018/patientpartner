# 英语教学视频生产工具

自动化英语教学视频生产全流程，从教学内容输入到即梦平台视频生成。

## 功能特性

- ✨ 自动拆分教学内容成15秒视频段落
- 📝 生成符合Seedance格式的视频提示词
- 👤 智能匹配人物卡和服装
- 🎨 生成专业封面图提示词
- 🚀 支持即梦平台自动化操作

## 使用方法

### 基本使用

当需要制作英语教学视频时，使用以下触发词：

```
制作英语教学视频
生成英语视频
英语教学视频生产
批量生成英语视频
英语短视频制作
```

### 输入格式

提供教学内容，可以是以下格式：

**格式1：教学文档**
```
主题：出国问路
句子1：Where is the hotel? 酒店在哪？
句子2：Where is the toilet? 洗手间在哪？
句子3：Where is the subway? 地铁站在哪？
```

**格式2：表格形式**
| 主题 | 英文句子 | 中文翻译 |
|------|----------|----------|
| 出国问路 | Where is the hotel? | 酒店在哪？ |
| ... | ... | ... |

**格式3：已有脚本**
直接提供需要转换的脚本内容

### 工作流程

1. **分析内容** - 识别主题和英文句子
2. **生成视频提示词** - 按15秒结构拆分，包含口播台词
3. **选择人物卡** - 从库存中选择最合适的服装
4. **生成封面提示词** - 制作吸引人的封面
5. **即梦平台操作** - 生成视频和封面（可选自动操作）

## 关键规则

### 视频提示词规则

1. **时长/比例不写进Prompt**
   - ❌ 错误：`15秒教学短视频，竖屏9:16，世界地图背景...`
   - ✅ 正确：`世界地图背景，暖色调明亮风格。0-3秒：...`

2. **必须写口播台词**
   - ❌ 错误：`字幕显示"Where is the hotel?"`
   - ✅ 正确：`主播清晰说："Where is the hotel?"，停顿，说"酒店在哪？"，字幕同步出现`

3. **主播设定**
   - 40岁左右，知性优雅
   - 语速缓慢，口型清晰
   - 英文后停顿，再说中文

### 封面图规则

1. **必须包含中文标题**
2. **关键词高亮**：数字用橙色，效果词用金色
3. **底部1/3留白**给标题
4. **防文字污染**：添加"画面干净没有任何额外文字水印"

## 即梦平台参数

### 视频生成
- **模型**：Seedance 2.0
- **比例**：9:16
- **时长**：15秒
- **批次**：每批5个

### 封面生成
- **模型**：Seedream 5.0
- **比例**：9:16
- **数量**：每主题2-3张

## 注意事项

1. 提示词中不要包含时长和比例信息
2. 必须明确写出主播口播的具体台词
3. 人物卡选择要匹配教学场景
4. 封面图要有吸引人的标题设计
5. 即梦平台操作时每批次不要超过5个视频
