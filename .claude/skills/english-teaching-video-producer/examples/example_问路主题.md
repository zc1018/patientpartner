# 示例：出国问路主题视频生产

## 输入内容

**教学主题**：出国问路
**英文句子**：
1. Where is the hotel? / 酒店在哪？
2. Where is the toilet? / 洗手间在哪？
3. Where is the subway? / 地铁站在哪？

## Phase 1: 视频提示词

```
世界地图背景，暖色调明亮风格。

0-3秒：特写镜头，知性优雅的女主播（40岁左右）对着镜头微笑点头，清晰说道："出国玩想问路，不用紧张。记住这几句，走到哪都不怕。"画面左下角同步浮现封面标题"出国问路就这3句超管用"。

3-6秒：中景，主播手势指向身旁虚拟世界地图，清晰说："Where is the hotel?"，停顿，说"酒店在哪？"，画面右侧同步显示中英文字幕"Where is the hotel? 酒店在哪？"。

6-9秒：主播手势变换指向另一位置，清晰说："Where is the toilet?"，停顿，说"洗手间在哪？"，字幕同步切换为"Where is the toilet? 洗手间在哪？"，背景地图元素对应变化为洗手间图标。

9-12秒：主播继续指向，清晰说："Where is the subway?"，停顿，说"地铁站在哪？"，字幕同步切换为"Where is the subway? 地铁站在哪？"，背景地图元素相应变化为地铁站图标。

12-15秒：主播微笑总结说："Where is + 地点，零基础也能轻松会。学会了就点个小爱心，关注我！"，双手比心，右下角浮现"关注"提示。

全程要求：主播口型清晰精准，英文发音标准缓慢，每句说完后停顿；中英文字幕与口播完全同步；画面干净，光线柔和自然。
```

## Phase 2: 人物卡选择

**选择**：`人物_米色衬衫外套_卡其长裤.png`

**理由**：
- 服装风格知性专业，适合教学场景
- 米色+卡其配色温暖亲切
- 世界地图背景与该人物卡气质匹配
- 通用性强，适合问路主题

**提示词中的服装描述**：
```
知性优雅的女主播，40岁左右，穿米色衬衫外套和卡其长裤
```

## Phase 3: 封面图提示词

```
温馨教育类短视频封面，一位年轻的欧美女性，20多岁，金发，甜美笑容，穿米色衬衫外套和卡其长裤，站在世界地图墙面前，背景是暖色调的地图元素和柔和光线。她正手指向地图做讲解姿态，表情自然活泼亲切。

画面下方大标题文字："出国问路只会比划？3句万能英语走到哪都不怕"，其中"3句万能"用橙色高亮，"走到哪都不怕"用金色强调。

暖黄色调，照片级写实风格，Instagram生活方式摄影，光线柔和，人物清晰突出，底部1/3留白适合叠加标题，画面干净没有任何额外文字水印。
```

## Phase 4: 即梦平台操作步骤

### 生成视频

1. 打开即梦 https://jimeng.jianying.com
2. 选择"视频生成" → "Seedance 2.0"
3. 设置参数：
   - 比例：9:16
   - 时长：15秒
4. 上传人物卡：`人物_米色衬衫外套_卡其长裤.png`
5. 粘贴上述视频提示词
6. 点击生成
7. 等待结果并下载

### 生成封面

1. 选择"图片生成" → "Seedream 5.0"
2. 设置参数：
   - 比例：9:16
3. 粘贴上述封面图提示词
4. 生成2-3张备选
5. 选择最佳图片下载
