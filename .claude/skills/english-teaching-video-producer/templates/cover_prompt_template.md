# 英语教学视频封面图提示词模板

## 标准模板

```
温馨教育类短视频封面，一位年轻的欧美女性，20多岁，{发色}，甜美笑容，穿{服装}，坐在{场景}里，背景是暖色调{背景元素}和{光线}。她正向镜头{动作}，表情自然活泼亲切。

画面下方大标题文字："{中文标题}"，其中"{关键词1}"用橙色高亮，"{关键词2}"用金色强调。

暖黄色调，照片级写实风格，Instagram生活方式摄影，光线柔和，人物清晰突出，底部1/3留白适合叠加标题，画面干净没有任何额外文字水印。
```

## 变量选择表

### 发色选择
- 金发（温暖亲切，适合大多数场景）
- 棕发（自然日常，适合商务场景）

### 服装选择（根据人物卡）
| 人物卡 | 服装描述 |
|--------|----------|
| 人物_米色衬衫外套_卡其长裤 | 米色衬衫外套和卡其长裤 |
| 人物_米色衬衫_咖啡馆场景 | 米色衬衫 |
| 人物_浅蓝长裙_牛仔马甲 | 浅蓝色长裙配牛仔马甲 |
| 人物_白色无袖上衣_黑色长裙_提包 | 白色无袖上衣和黑色长裙 |
| 人物_白色蕾丝上衣_黑色开叉长裙 | 白色蕾丝上衣和黑色开叉长裙 |
| 人物_米色高领上衣_蓝色碎花长裙 | 米色高领上衣和蓝色碎花长裙 |
| 人物_绿色旗袍_中式改良 | 绿色改良旗袍 |
| 人物_蓝色衬衫_黑色包臀裙 | 蓝色衬衫配黑色包臀裙 |

### 场景选择
| 教学主题 | 推荐场景 |
|----------|----------|
| 问路/导航 | 明亮温馨的咖啡厅 |
| 餐厅/点餐 | 温馨西餐厅 |
| 购物/问价 | 时尚购物街 |
| 酒店/入住 | 高端酒店大堂 |
| 机场/交通 | 机场贵宾休息室 |
| 日常交流 | 简约居家环境 |

### 背景元素
- 木质装潢和绿植
- 温馨布置和柔和灯光
- 书架和装饰物
- 大窗户和自然光

### 动作选择
- 友好挥手打招呼
- 微笑示意
- 亲切点头
- 温暖拥抱姿势

## 标题高亮规则

**关键词选择优先级**：
1. 数字（如"3句"、"4句"）→ 用橙色高亮
2. 结果/效果词（如"走到哪都不怕"、"轻松点餐"）→ 用金色强调
3. 核心动词（如"万能"、"必备"）→ 根据语境选择
