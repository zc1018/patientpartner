# 英语教学视频提示词模板

## 标准结构（15秒）

```
{场景背景}，{氛围描述}。

0-3秒：特写镜头，知性优雅的女主播（40岁左右）对着镜头微笑点头，清晰说道："{中文引入语}"，画面左下角同步浮现封面标题"{标题}"。

3-6秒：中景，主播{手势1}，清晰说："{英文句1}"，停顿，说"{中文翻译1}"，画面右侧同步显示中英文字幕"{英文句1} {中文翻译1}"。

6-9秒：主播{手势2}，清晰说："{英文句2}"，停顿，说"{中文翻译2}"，字幕同步切换为"{英文句2} {中文翻译2}"。

9-12秒：主播{手势3}，清晰说："{英文句3}"，停顿，说"{中文翻译3}"，字幕同步切换为"{英文句3} {中文翻译3}"。

12-15秒：主播微笑总结说："{总结语}，学会了就点个小爱心，关注我！"，双手比心，右下角浮现"关注"提示。

全程要求：主播口型清晰精准，英文发音标准缓慢，每句说完后停顿；中英文字幕与口播完全同步；画面干净，光线柔和自然。
```

## 场景背景匹配表

| 教学主题 | 推荐场景背景 | 氛围描述 |
|----------|-------------|----------|
| 问路/导航 | 世界地图背景 | 暖色调明亮风格 |
| 餐厅/点餐 | 温馨西餐厅/餐桌背景 | 暖黄色调，食欲氛围 |
| 购物/问价 | 购物街/商店橱窗背景 | 明亮热闹氛围 |
| 酒店/入住 | 酒店大堂/前台背景 | 温馨专业氛围 |
| 机场/交通 | 机场值机柜台/候机厅背景 | 旅行氛围 |
| 日常交流 | 干净纯色背景（浅蓝/米色） | 简洁清新风格 |
| 紧急情况 | 医院/药店背景（柔和处理） | 安全导向氛围 |

## 手势推荐

| 英文句子类型 | 推荐手势 |
|-------------|----------|
| Where is... | 手指指向、询问手势 |
| I want... | 拿取手势、点餐手势 |
| How much... | 询问价格手势、摊开双手 |
| Thank you | 双手合十感谢、点头 |
| Check in/out | 递证件手势、离开手势 |
| I don't... | 摆手拒绝、摇头手势 |

## 标题设计公式

```
{场景痛点}？{数字}句{解决方案}{效果}
```

**示例**：
- 出国问路只会比划？3句万能英语走到哪都不怕
- 餐厅点餐只会指菜单？3句万能英语轻松点餐
- 住酒店说不出需求？4句英文入住退房全搞定
- 听不懂只会点头？2句礼貌用语化解尴尬
