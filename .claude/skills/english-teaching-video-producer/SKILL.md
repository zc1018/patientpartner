# 英语教学视频生产工具

## 概述

自动化英语教学视频生产全流程工具，从中英文教学内容输入到即梦平台视频生成，一站式完成视频拆分、提示词生成、人物卡选择、封面制作和平台操作。

## 适用场景

- 旅游英语、商务英语、日常口语等教学短视频制作
- 批量生成系列英语教学视频（15秒/条）
- 需要统一人物形象和风格的视频系列
- 零基础英语教学视频内容

## 核心工作流程

### Phase 1: 分析教学内容

**输入格式**：
- 教学文档（Markdown/文本）
- 教学内容列表（主题 + 英文句子 + 中文翻译）
- 已有脚本需要转换

**处理步骤**：
1. 识别教学主题
2. 提取核心英文句子（2-3句/主题）
3. 确定场景背景（根据主题匹配）

### Phase 2: 拆分视频段落

**标准结构**（15秒/条）：

```
0-3秒：开场引入
- 主播微笑出镜
- 中文引入语（说明主题重要性）
- 封面标题浮现

3-12秒：教学内容（分3段，每段3秒）
- 每段：英文句子 + 中文翻译
- 配合手势和场景演示

12-15秒：结尾引导
- 总结语
- 关注引导
- 互动提示
```

### Phase 3: 生成视频提示词

**必须遵守的规则**：

1. **时长/比例不写进Prompt**
   - ❌ 不要写："15秒教学短视频，竖屏9:16，..."
   - ✅ 直接从内容开始："世界地图背景，暖色调明亮风格。0-3秒：..."

2. **必须明确口播台词**
   - ❌ 不要只写："字幕显示'Where is the hotel?'"
   - ✅ 必须写："主播清晰说：'Where is the hotel?'，停顿，说'酒店在哪？'，字幕同步出现"

3. **主播统一设定**
   - 年龄：40岁左右，知性优雅
   - 语速：缓慢清晰，口型精准
   - 每句英文后停顿，再说中文翻译

**视频提示词模板**：

```
{场景背景}，{氛围描述}。

0-3秒：特写镜头，知性优雅的女主播对着镜头微笑点头，清晰说道："{中文引入语}"，画面左下角同步浮现封面标题"{标题}"。

3-6秒：中景，主播{手势}，清晰说："{英文句1}"，停顿，说"{中文翻译1}"，画面右侧同步显示中英文字幕"{英文句1} {中文翻译1}"。

6-9秒：主播{手势}，清晰说："{英文句2}"，停顿，说"{中文翻译2}"，字幕同步切换为"{英文句2} {中文翻译2}"。

9-12秒：主播{手势}，清晰说："{英文句3}"，停顿，说"{中文翻译3}"，字幕同步切换为"{英文句3} {中文翻译3}"。

12-15秒：主播微笑总结说："{总结语}，学会了就点个小爱心，关注我！"，双手比心，右下角浮现"关注"提示。

全程要求：主播口型清晰精准，英文发音标准缓慢，每句说完后停顿；中英文字幕与口播完全同步；画面干净，光线柔和自然。
```

### Phase 5: 生成封面图提示词

**封面图模板**：

```
温馨教育类短视频封面，一位年轻的欧美女性，20多岁，{发色}，甜美笑容，穿{服装}，坐在{场景}里，背景是暖色调{背景元素}和{光线}。她正向镜头{动作}，表情自然活泼亲切。

画面下方大标题文字："{中文标题}"，其中"{关键词1}"用橙色高亮，"{关键词2}"用金色强调。

暖黄色调，照片级写实风格，Instagram生活方式摄影，光线柔和，人物清晰突出，底部1/3留白适合叠加标题，画面干净无任何额外文字水印。
```

**标题设计公式**：
```
{场景痛点}？{数字}句{解决方案}{效果}
```

**示例**：
- "出国问路只会比划？3句万能英语走到哪都不怕"
- "餐厅点餐只会指菜单？3句万能英语轻松点餐"
- "住酒店说不出需求？4句英文入住退房全搞定"

### Phase 6: 即梦平台操作

**使用 web-access skill 操作浏览器**：

1. **登录即梦**
   - 访问 https://jimeng.jianying.com
   - 使用用户账号登录（需用户提前登录或提供登录方式）

2. **生成视频（Seedance 2.0）**
   - 选择 "视频生成" → "Seedance 2.0"
   - 比例选择：9:16
   - 时长选择：15秒
   - 每批次生成5个视频（避免过多导致等待时间过长）
   - 操作步骤：
     1. 上传对应人物卡作为参考图
     2. 粘贴视频提示词
     3. 点击生成
     4. 记录生成任务ID或等待结果

3. **生成封面图（Seedream 5.0）**
   - 选择 "图片生成" → "Seedream 5.0"
   - 比例选择：9:16
   - 粘贴封面图提示词
   - 每主题生成2-3张备选

**批量处理策略**：
- 视频：每5个为一批，分批生成
- 封面：每主题2-3张，供选择
- 记录所有生成任务的输出链接

## 完整执行流程

当用户输入教学内容后，按以下步骤执行：

1. **读取并分析输入内容**
   - 识别教学主题数量
   - 提取每个主题的英文句子
   - 确定场景分类

2. **为每个主题生成**：
   - 视频提示词（按Phase 3模板）
   - 封面图提示词（按Phase 5模板）
   - 人物卡选择（按Phase 4匹配）

3. **整理生成文档**：
   - 所有视频提示词汇总
   - 所有封面提示词汇总
   - 人物卡选择清单

4. **生成即梦操作指南**：
   - 详细的浏览器操作步骤
   - 每批次生成清单
   - 人物卡上传顺序

5. **（可选）使用 web-access 自动操作**：
   - 登录即梦平台
   - 按批次生成视频和封面
   - 记录生成结果

## 注意事项

### 视频提示词禁忌
- 不要在prompt里写"15秒"、"9:16"，这些在平台配置
- 不要只写字幕，必须写主播口播台词
- 英文句子后必须加停顿再说中文

### 人物卡使用
- 必须先查看 `/Users/xdf/Documents/剧本/视频人物卡/` 目录下实际存在的文件
- 根据主题场景选择最合适的人物卡
- 记录每个人物卡的完整路径

### 封面图要求
- 必须包含中文标题文字描述
- 关键词高亮（橙色/金色）
- 底部1/3留白给标题
- 添加防文字污染指令

### 平台操作
- 视频使用 Seedance 2.0 模型
- 封面使用 Seedream 5.0 模型
- 比例统一 9:16
- 每批次最多5个，避免超时
